# Pour accroitre le SEO, l'accessibilté de la page et la performence de mon application

J'ai du : 
- Ajouter des balises meta pour améliorer la visibilité de mon application sur les moteurs de recherche.
- Ajouter des différents attributs sur plusieurs balises comme des alt, des loading="lazy" et des decoding="async" sur les images pour améliorer le chargement de ma page.
- Ajouter des tabindex sur plusieurs balises pour améliorer l'accessibilité de ma page.
- Vérifier la hiérarchisation de ma page avec les balises h1, h2, h3, etc.
- Ajouter des balises Suspense pour améliorer la performance de ma page.
- Rajouter du texte dans les boutons qui contenaient uniquement une icone.
